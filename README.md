# Nymbl UI

## Setup


### Install dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Build Style Guide

```
npm run build
```

### Build UI bundle

```
npm run build:bundle
```

## Tests

### Run your unit tests

```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
