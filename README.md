# Nymbl UI

https://nymblscience.github.io/nymbl-ui

## Development

### Install dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Build Style Guide

```
npm run build
```

### Build UI bundle

```
npm run build:bundle
```

## Tests

### Run your unit tests

```
npm run test:unit
```

## Deploy to Github Pages

### Install `gh-pages` globally

```
npm install -g gh-pages
```

### Build & Deploy

```
npm run deploy
```
